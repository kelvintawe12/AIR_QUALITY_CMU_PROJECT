# Hardware Connections

## Pinout
- **MQ135 (Air Quality):** Analog Out → Arduino A0
- **MQ7 (CO):** Analog Out → Arduino A1
- **DHT22 (Temp/Humidity):** Data → Arduino Digital 2
- **Power:** All sensors to 5V and GND
