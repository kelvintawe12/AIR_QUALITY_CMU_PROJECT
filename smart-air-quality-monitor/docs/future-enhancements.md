# Future Enhancements

- **Mobile App:** Native iOS/Android app for remote monitoring and push notifications.
- **Cloud Logging:** Sync sensor data to a cloud database (Firebase, AWS, etc.) for historical analytics and remote access.
- **Custom PCB:** Design a custom PCB for compact, reliable hardware integration.
- **More Sensors:** Add PM2.5, eCO2, TVOC, dust, and other environmental sensors.
- **User Profiles:** Multi-user support, personalized alerts, and analytics.
- **Advanced ML:** Deploy deep learning models for more accurate risk prediction.
- **OTA Updates:** Over-the-air firmware updates for Arduino.
- **API Integrations:** Connect with smart home systems (HomeKit, Alexa, Google Home).
- **Data Export:** Download CSV/JSON of historical data from the dashboard.
- **Admin Dashboard:** System health, logs, and device management for admins.
